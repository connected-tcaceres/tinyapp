## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- cookie-session
- connect-flash
- method-override

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` or `npm start` command.
